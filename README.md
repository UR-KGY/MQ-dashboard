# 📊 Infrastructure Monitoring Dashboard

Nuxt 4와 Prometheus를 활용한 실시간 인프라 모니터링 대시보드 프로젝트입니다.

## 🛠 Tech Stack
* **Frontend**: Nuxt 4 (Minimal Template)
* **Backend/Runtime**: Node.js v22.22.2 LTS
* **Monitoring**: Prometheus v3.5.2
* **Infrastructure**: Docker, Windows 환경

## 📂 Project Structure
* `frontend/`: Nuxt 4 기반의 대시보드 웹 어플리케이션
* `backend/`: 데이터 처리를 위한 Node.js 서버 (예정)
* `infra/`: Prometheus 설정 및 로컬 인프라 구성 파일
